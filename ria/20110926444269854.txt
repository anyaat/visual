Американские биологи приспособили светящуюся кишечную палочку для шифровки и передачи информации при помощи этих бактерий они закодировали сообщения которые скрываются от взора до определенного времени и самоуничтожается после прочтения либо требуют для проявки определенных реактивов говорится в статье опубликованной в журнале
Дэвид Уолт из Университета Тафтса США и его коллеги назвали свое изобретение бактериальный от английского стеганография при помощи печатных микробных матриц Стеганографию можно считать старшей сестрой криптографии при стеганографировании сообщения скрывается не только его содержимое но и сам факт его существования
Авторы статьи вырастили семь различных штаммов кишечной палочки каждый из которых вырабатывал свой тип светящегося вещества
Ученые подготовили по два варианта каждого штамма первый выделял флуоресцирующие вещества через некоторое время а второй сразу после переноса бактерий на питательную среду
Затем биологи подготовили специальную бумагу матрицу из микроскопических лунок с питательной средой
В каждую такую емкость можно поместить каплю с одним штаммом бактерии
Пара подобных лунок задает различных сочетаний символов-микроорганизмов чего достаточно для кодирования всех букв латинского алфавита знаков препинания и цифр
Для проверки своего изобретения авторы статьи напечатали первое закодированное сообщение из букв-долгоиграющих бактерий
Оно гласило это биокодированное сообщение из лаборатории уолта при университете тафтс год
Уолт и его коллеги смогли прочитать свой опус только через два дня после нанесения чернил за это время бактерии размножились и выделили достаточное количество флуоресцирующих веществ
Время ожидания можно значительно сократить до восьми часов если использовать второй быстрый тип бактерий
Для этого читатель должен нанести на лист с сообщением раствор вещества ускорителя который заставит бактерии работать сверхурочно и вырабатывать большие количества краски
Биологи отмечают что сложность шифра можно увеличить если вставить в геном бактерии маркирующий ген с помощью которого можно отличить значимые микроорганизмы-буквы от фонового бактериального шума
Для демонстрации работы этой идеи авторы статьи вставили гены устойчивости к двум антибиотикам канамицину и ампицилину в два набора цветных бактерий
Ученые напечатали два разных сообщения на одной и той же питательной бумаге при помощи взломоустойчивых микроорганизмов
Такой лист можно обработать одним из двух лекарств и прочитать только одно из двух сообщений
Таким образом если читатель не обладает правильным шифром-антибиотиком он никогда не узнает то что скрывают бактерии
Кроме того ученые разрабатывают еще одну серию штаммов кишечной палочки которые можно использовать для печати специальных сообщений самоуничтожающихся после прочтения
Эти бактерии будут вырабатывать флуоресцирующие вещества которые меняют свой цвет через некоторое время
Каждая буква кодируется сочетанием из двух бактерий и если одна из них изменит свой знак сообщение будет безнадежно испорчено
